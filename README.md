# Rudventure
터미널에서'만' 작동하는 실시간 로그라이크 RPG   
   
## 시작하기에 앞서   
   실행이 되지 않을 때는 파이썬 3.13.0이 설치되어 있는지 확인해주세요.

## 플레이 방법   
### Mac   
```
cd desktop; git clone https://github.com/Daho0980/Rudventure.git
```
을 입력 후 'runGame.command'를 클릭하여 실행   

### Windows   
```
현재 지원 불가
```
