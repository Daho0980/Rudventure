# Rudventure

## 게임 소개
그저 로그라이크류 터미널 게임   
   
## 플레이 방법   
### Mac   
```
git clone https://github.com/Daho0980/Rudventure.git
cd Rudventure
python3 runGame.py
```
- 이와 같이 다운로드 시 게임 파일은 (사용자 이름)/Rudventure 에 위치해 있음

### Windows   
```
Mac과 동일
```
아직 최적화는 되지 않아 버벅거림이 심할 수 있음
