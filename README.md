# Rudventure

## 게임 소개
그저 평범하게 실행되는 걸 허락하지 않는 로그라이크 게임   
   
## 플레이 방법   
### Mac   
```
cd desktop
git clone https://github.com/Daho0980/Rudventure.git
```
을(를) 입력 후 'run_for_mac.sh'를 클릭하여 실행   

### Windows   
아쉽게도,   
```
아직 만들어지지 않았습니다!
```
